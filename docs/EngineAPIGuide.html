Coding Standards

Naming Conventions:
  • Type aliases: T-prefix (TByte, TInt, TFloat3)
  • Template parameters: T-prefix (TNumber, TVec)
  • Concepts: C-prefix (CLockable, CNext)
  • Enum classes: E-prefix (ELogLevel)
API Conventions:
  • Non-throwing functions: noexcept
  • Value-returning getters: [[nodiscard]]
  • Single-arg constructors: explicit
  • RAII for resource management (ScopedLock, ScopedTime, AllocatorScope)
  • Assertions in debug builds; FatalAssert always active
