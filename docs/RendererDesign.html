1. High-Level Data Flow Architecture

The renderer implements a strictly decoupled, data-oriented hybrid pipeline through four distinct phases: Extract → Prepare → Render Graph → Submit.

2. HLSL-Centric Bindless Architecture

The RHI uses a unified 32-bit handle approach where shaders access textures and buffers via indices into global arrays. HLSL (Shader Model 6.6+) serves as the single source of truth, cross-compiled to DXIL, SPIR-V, and MSL.

4. Stateless Command Buckets & Multi-threading




Thread-Local Command Bucket


struct ThreadLocalCommandBucket {
    volatile std::atomic<uint32_t> builtCount = 0;  // Atomic counter
    
protected:
    void RecordGraphicsCommand(DrawSortKey& key, DrawCommand& cmd) {
        // Each thread has its own bucket - no mutex during recording!
        size_t idx = ++builtCount % NUM_BUCKETS;
        buckets[idx].sortKey = key;
        buckets[idx].meshHandle = cmd.meshHandle;
    }
};

// At frame end: parallel radix sort across all threads
void AggregateAndRadixSortParallel(std::vector<DrawCommand&> out) {
    // Gather from thread-local buckets into flat array
    // Radix sort by 64-bit key (zero contention!):
    std::sort(out.begin(), out.end(), 
      [](const Draw&a, const Draw&b){ return a.sortKey < b.sortKey; });
}

7. Critical Edge Cases





Metal Tier 2 Limits: Descriptor Partitioning


inline void DescriptorsSplitIntoPages(BindlessDescriptorSet& set, uint32_t maxBindingsPerPass = 16) {
    const uint32_t MAX_TEXTURES_PER_PAGE = 64u; // Metal hardware limit
    
    uint32_t currentPageIdx = 0;
    for (uint32_t i = 0; i < MAX_BINDLESS纹理; ) {
        set.textures[currentPageIdx] = SetNextDescriptorBatch(set.textures, i, MAX_TEXTURES_PER_PAGE);
        i += MAX_TEXTURES_PER_PAGE;
        
        // Metal Tier 2 check: split into additional argument buffers
        if (IsBelowMetalTier1()) {
            MTLBuffer newBuffer = AllocateMTLArgumentBufferFromPool(MAX_TEXTURES_PER_PAGE);
            currentPageIdx++;
        }
    }
}








Temporal History Aliasing Prevention


// Preserve motion vectors and history buffers across frames
struct TemporalTextureMetadata {
    uint32_t generation;                  // Frame counter for alias detection
    vec2 motionVector[MAX_HISTORIES];      // For TAA/DLSS motion vectors
    bool      isHistoryCritical = false;   // Prevents premature reclamation
    
    void EnsureTemporalConsistency() const {
        if (isHistoryCritical && generation != GetCurrentFrameGen()) {
            CopyMotionVectorsToHistoryBuffer(motionVector);
        }
    }
};
